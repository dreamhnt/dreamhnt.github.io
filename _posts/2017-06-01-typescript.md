# 설치 & 사용법

사용해야되니 설치를 한다.

`npm install -g typescript`

확장자는 **.ts** 이고, 설치를 하면 tsc 명령어를 사용할 수 있다. ts 파일을 js 파일로 변환하려면 명령어를 사용하면 된다.

`tsc test.ts`

코드 확인 방법은

- ts 파일을 js로 변환 후 html 파일 만들어서 확인
- [Playground](https://www.typescriptlang.org/play/index.html)
